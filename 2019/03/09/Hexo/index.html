安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。

More info: Hexp

https://github.com/CedarChennn/CedarChennn.github.io.git

$ hexo -g #在修改配置文件后需要重新执执行此命令创建生成静态文件,之后部署网站修改的配置文件才会生效
$ hexo clean #清除缓存文件 (db.json) 和已生成的静态文件 (public)。
